## ✦ Overview

A performance-focused portfolio featuring scroll-triggered animations, an interactive vinyl record resume section, magnetic cursor effects, and a fully responsive design that works seamlessly across all devices.

## ✦ Tech Stack

| Category | Technologies |
|---|---|
| **Framework** | Next.js 16, React 19 |
| **Language** | TypeScript (strict mode) |
| **Styling** | Tailwind CSS 4 |
| **Animations** | Framer Motion, GSAP + ScrollTrigger |
| **Typography** | Bebas Neue, DM Sans, JetBrains Mono |
| **Deployment** | Vercel |

## ✦ Sections

- **Hero** — Animated intro with text scramble effects and parallax
- **About** — Personal background with scroll-reveal animations
- **Work** — Project showcase with expandable cards and GitHub links
- **Skills** — Infinite marquee with categorized tech stack
- **Resume** — Interactive vinyl record with download/view CTA
- **Contact** — Animated form with real-time validation

## ✦ Key Features

- GSAP ScrollTrigger-powered scroll animations
- Framer Motion entrance transitions and micro-interactions
- Magnetic button hover effects
- Custom cursor with context-aware states
- Smooth scrolling with Lenis
- Dark/light section rhythm for visual depth
- Fully responsive — mobile to ultrawide
